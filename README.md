# 🏛️ Temple Dash

> *Run. Dodge. Survive. One more run.*

A fast-paced browser-based endless runner where you sprint through ancient temple ruins — dodging traps, grabbing coins, and chasing your best score. Built with pure HTML, CSS & JavaScript. No engine. No frameworks. Just code.

---

## 🎮 Play Now

👉 **[Play Temple Dash](https://kazim-45.github.io/Temple-dash/temple_dash.html)** — runs instantly in your browser, no install needed.

---

## ✨ Features

- 🏃 **Endless running** — procedurally generated obstacles that keep every run fresh
- 🪤 **Obstacles & traps** — spikes, walls, and temple hazards that punish the careless
- 🪙 **Coins** — collect as many as you can mid-run
- 🏆 **High score leaderboard** — beat your best, claim the top spot
- ⚡ **Speed scaling** — the longer you survive, the faster it gets
- 🖥️ **Browser-native** — zero downloads, runs anywhere

---

## 🕹️ How to Play

| Key | Action |
|-----|--------|
| `Space` / `↑` | Jump |
| `↓` | Slide / Duck |
| `→` | Speed boost (if applicable) |

> On mobile, tap the screen to jump.

## 🛠️ Built With

```
HTML5       →  Structure & Canvas
CSS3        →  Styling & animations  
JavaScript  →  Game logic, physics, collision detection
```

No libraries. No game engine. Built from scratch.

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/kazim-45/Temple-dash.git

# Open in browser
cd temple-dash
open index.html
```

That's it. No build step, no dependencies.
